# Проект: Место
----------

## Описание
Учебный проект, позволяющий редактировать профиль и оставлять отметку 'нравится'
на понравившиеся новости.

### Технологии
Адаптивная вёрстка, БЭМ, формы, флекс и грид вёрстка.
